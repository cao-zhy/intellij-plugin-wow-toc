# WoW TOC

使 IntelliJ IDEA 支持魔兽世界插件的 .toc 文件。

## 功能

- 新建 .toc 文件

- 编辑新建 .toc 文件的模板

- 语法高亮

- 自定义代码颜色设置

- 标签名和引用文件的注解

- 快速修复

- 官方标签名提示

- 引用文件名提示

- 打开引用文件

- 重命名引用文件

- 格式化代码

- 自定义代码风格设置

- 注释

## 安装

在 Plugin Marketplace 上搜索 “wow toc” 安装，或在[插件主页](https://plugins.jetbrains.com/plugin/13375-wow-toc/)下载后本地安装。
